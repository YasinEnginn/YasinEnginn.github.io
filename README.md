# Yasin Engin Expert - Kişisel Web Sitesi

GitHub Pages üzerinde barındırılan kişisel web sitesi ve portföy projesi.

## URL
[https://yasinenginexpert.github.io](https://yasinenginexpert.github.io)

## Özellikler
- **Responsive Tasarım:** Mobil ve masaüstü uyumlu.
- **Temiz Arayüz:** Sade ve modern görünüm.
- **Hızlı Yükleme:** Optimize edilmiş statik HTML/CSS.

## Kurulum (Lokal)
1. Repoyu klonlayın:
   ```bash
   git clone https://github.com/YasinEnginExpert/yasinenginexpert.github.io.git
   ```
2. Klasöre girin:
   ```bash
   cd yasinenginexpert.github.io
   ```
3. `index.html` dosyasını tarayıcınızda açın.

## Teknoloji Stack
- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- Vanilla JavaScript
